# io_bto

`Genesis (Build To Order) (.json)` import addon for [Blender](http://www.blender.org/).

## Supported formats
| Extension | Description           | Can import |
|-----------|-----------------------|:----------:|
| .json     | Source models         | Yes        | 

## Scope & Limitations

This addon imports only Genesis (Build To Order) JSON files that you have saved yourself.
